# Método de Descenso del Gradiente
**Equipo Laplace**
A continuación se adjunta el algoritmo de descenso del gradiente para la busqueda de mínimos locales de una función de dos variables desarrollado para la Situación Problema 'Redes Neuronales y Ecuaciones Diferenciales' de la materia MA1033 en el lenguaje de programación MATLAB.

## Secciones Programa

- Variables a Modificar
  -  Variables para el cálculo de mínimos a partir del descenso del gradiente
  -  Variables para la gráficación de la función y el punto cálculado
-  Preparación para el algoritmo del Descenso del Gradiente
-  Ciclo del Descenso del Gradiente
  -  Cálculo del siguiente punto en base al descenso del gradiente
-  Impresión y graficación de los resultados

## Funcionamiento
El usuario debe sustituir las variables correspondientes dentro de la sección de **Variables a Modificar** y ejecutar el código. Este automaticamente dara el punto obtenido según el método de paro que se haya ejecutado y la gráficación tridimensional de la función y el punto final.
